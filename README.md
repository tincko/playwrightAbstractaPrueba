# Abstracta Playground UI Automation

This repository contains UI automation tests for the Demoblaze ecommerce site as part of the Abstracta selection process (or exercise).

## Project Structure

- **pages/**: Contains Page Object Model (POM) classes.
- **tests/**: Contains test specifications.
  - `part1_scraper.spec.ts`: Scrapes product data from the first 2 pages.
  - `part2_automation.spec.ts`: Automates purchase flow, signup, and login.
- **utils/**: Utility files.
- `scraped_products.txt`: Output file generated by Part 1.

## Prerequisites

- Node.js (v14+)
- npm

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Install Playwright browsers:
   ```bash
   npx playwright install
   ```

## Running Tests

To run all tests:
```bash
npx playwright test
```

To run a specific test file:
```bash
npx playwright test tests/part1_scraper.spec.ts
```

## Viewing Reports

Playwright generates an HTML report after execution.
```bash
npx playwright show-report
```

## Scraped Data

After running `part1_scraper.spec.ts`, the product data will be saved to `scraped_products.txt` in the root directory.
